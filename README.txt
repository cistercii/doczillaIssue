# Конкатенация текстовых файлов

В данном проекте происходит поиск рекурсивный поиск текстовых файлов, лежащих в папке src/main/resource, сортировка
файлов по имени и конкатенация содержимого в результирующий файл result.txt

### Сборка и Запуск (для сборки нужен Maven)

1. Скопировать проект.
2. Перейти в папку с проектом и собрать его командой:
    mvn compile
3. Добавить в папку src/main/resources файлы или папки с файлами, в которых будет производится поиск.
4. Запустить приложение командой:
    mvn exec:java -Dexec.mainClass="Main"
